## 임재형

### 프로젝트
#### 1. 챔버 과온 진단 및 안전 복구 (26.04.01 ~ 26.04.27)
STM32와 PI 제어를 기반으로 챔버 과온 감지, 2단계 경보(주의·위험), 히터 자동 차단, 복구 조건 검증까지의 알람 진단 및 안전 복구 시스템을 구축. 냉각팬 미동작 고장을 직접 재현해 제어선 단선으로 원인을 특정했고, 5개 검증 테스트를 100% 통과시켰으며 7종 기술 문서를 작성함

#### 2. 쿠스토랑 (23.12.01 ~ 25.08.31)
건국대 인근 맛집 순위·추천 서비스를 기획 및 개발. PM과 백엔드 개발을 맡아 기술스택 선정, API 개발, 운영을 수행했으며 출시 직후 2일간 활성 사용자 4,000명을 확보하고 1년간 MAU 1,000을 유지함.

#### 3. O(1) 스케줄러 개발 (24.09.01 ~ 24.10.30)
ghOSt 기반 사용자 공간 O(1) 스케줄러 개발 프로젝트를 주도. 커널-에이전트 간 동기화 문제를 분석·해결해 12종 통합 테스트를 100% 통과시켰고, 코드와 실행 가이드를 GitHub에 정리함

#### 4. UFFS 파일시스템 개발 (24.11.01 ~ 24.12.31)
FUSE 기반 UFFS 파일시스템 개발 프로젝트를 주도. 로그 분석, hexdump, 콜백 추적으로 데이터 입출력 문제를 해결해 41개 I/O 통합 테스트를 100% 통과함

#### 5. 찾아줘 홈즈 (24.02.01 ~ 24.12.31)
졸업프로젝트로 챗봇 대화를 통해 개인 맞춤화 부동산을 추천해주는 앱을 개발함. 2024년도 건국대학교 졸업프로젝트 우수상을 받음.
